# Semana 1 — Cap 1 inicio: FVVR conceptos, límites, diferenciabilidad

## Sesiones

- **S1** — Conceptos básicos f: R → R^n. Dominio, rango, traza, gráfica. Operaciones (suma, diferencia, etc.). Esbozo de traza.
- **S2** — Límites: pto acumulación, pto aislado, def límite, teoremas, álgebra. Continuidad: def, teoremas.
- **S3** — Diferenciabilidad: def como límite + aprox por T: R → R^n. Vector tangente, velocidad, rapidez, recta tangente. Dif → cont. Álgebra derivadas. Regla cadena.
