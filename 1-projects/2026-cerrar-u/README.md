# 2026-cerrar-u — aprobar cálculo avanzado (trika) + base de datos (bika)

## Stakes

- **Cálculo avanzado**: 3ra matrícula. Si jala = **suspensión 1 año**. No negociable.
- **Base de datos**: 2da matrícula. Riesgo medio. Aprobar con margen.

Si Cálculo se aprueba y BD se jala: pérdida ciclo (BD), pero carrera sigue.
Si Cálculo se jala: 1 año fuera + replanteo carrera.

→ **Cálculo > BD > todo lo demás** durante 8 semanas.
